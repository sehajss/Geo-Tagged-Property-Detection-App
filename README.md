# Geo-Tagged Property Detection App
An Android application that detects property sale/rent signboards in real time using TensorFlow Lite and OCR, while tagging detections with GPS coordinates and generating trip-based reports.
## Features

- Real-time property signboard detection
- OCR text extraction from detected boards
- GPS-tagged detections
- Google Maps integration
- Trip tracking and history
- Detection reports in PDF and CSV format
- Offline TensorFlow Lite inference
- Detection database storage

## Tech Stack

- Kotlin
- Jetpack Compose
- TensorFlow Lite
- ML Kit OCR
- Google Maps SDK
- SQLite
- Android Studio

## How to Run

1. Clone the repository
2. Open the project in Android Studio
3. Sync Gradle files
4. Add your Google Maps API key in AndroidManifest.xml
5. Run the app on a physical Android device

## Permissions Used

- Camera
- Fine Location
- Coarse Location
- Storage/File Access
